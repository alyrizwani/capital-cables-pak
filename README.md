#  Capital Cables Pakistan — Premium Industrial Website

A modern, high-performance, and light-themed corporate website designed for **Capital Cables Pakistan (PVT Limited)**.  

This project transforms a traditional industrial web presence into a premium digital experience using refined typography, a clean architectural layout, and a fully responsive mobile-first design.

🌐 **Live Demo:** https://alyrizwani.github.io/capital-cables-pak/

---

## ✨ Overview

This website is built to reflect a strong industrial identity with a premium digital feel. It focuses on clarity, performance, and trust-building through minimal design and structured content flow.

---

## 🚀 Key Features

### 🎨 Premium Design System
- Elegant **Architectural Blue** theme (`#2C5D7F`)
- Clean, corporate-grade visual identity
- Balanced white space and structured layouts

### ✍️ Typography
- **Playfair Display** → Elegant serif for headings
- **Outfit** → Modern sans-serif for body text

### 📱 Fully Responsive
- Mobile-first layout
- Seamless experience across phones, tablets, and 4K displays
- Max-width container (`1280px`) for readability and consistency

### 🧩 Modern UI Components
- Floating **WhatsApp contact button**
- Glassmorphism-style navigation bar (scroll-based effect)
- Smooth scroll reveal animations
- Infinite scrolling brand/certification ticker

### ⚡ Performance Optimized
- Lightweight **vanilla JavaScript (ES6)**
- No heavy frameworks or libraries (no jQuery)
- Optimized CSS architecture for fast loading
